title New_Note_0.4

note over browser:
user gives input data in text field
and click submit button
the browser reloads the Notes page. 
The reload causes three more HTTP POST requests
end note

browser->server: HTTP POST https://fullstack-exampleapp.herokuapp.com/new_note
server-->browser: HTTP-status-302(reloaded)
browser->server: HTTP POST https://fullstack-exampleapp.herokuapp.com/notes
server-->browser: HTML-code
browser->server: HTTP POST https://fullstack-exampleapp.herokuapp.com/main.css
server-->browser: main.css
browser->server: HTTP POST https://fullstack-exampleapp.herokuapp.com/main.js
server-->browser: main.js

note over browser:
browser starts executing js-code
that requests JSON data from server 
end note

browser->server: HTTP POST https://fullstack-exampleapp.herokuapp.com/data.json
server-->browser: [{ content: "HTML is easy", date: "2019-05-23" }, ...]

note over browser:
browser executes the event handler
that renders notes to display
end note
